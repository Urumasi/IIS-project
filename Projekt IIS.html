Implementace

Pro implementaci jsme se rozhodli využít framework Flask. Backend našeho informačního systému tvoří Python a frontend se skládá z kombinace HTML a JavaScript. Pro databázi jsme zvolili Postgres.

Popis jednotlivých částí projektu
  • /app/admin: Přidávání nových uživatelů a schvalování navrhnutých předmětů - pouze pro uživatele s administrátorským oprávněním
  • /app/auth: Vytváření nových předmětů, aktualit a termínů, změna hesla - pro všechny přihlášené uživatele
  • /app/public: Přihlášení se a zobrazení všech kurzů - obecně přístupné
  • /app/data: Modely jednotlivýh prvků databáze
  • /app/static: CSS a JS soubory

Databáze

Instalace

Rozbalte všechny soubory. V root adresáři zadejte následující sekvenci příkazů:
  • sudo -iu postgres psql
  • CREATE DATABASE iis;
  • CREATE USER iis WITH PASSWORD 'password';
  • GRANT ALL PRIVILEGES ON DATABASE iis TO iis;
  • exit
  • flask db init
  • flask db migrate
  • flask db upgrade
  • python3 init_db.py
Poté spustítě informační systém přikazem:
  • flask run
Systém bude spuštěn na localhost:5000. Je možné tuto konfiguraci změnit upravením souboru config.py v adresáři app.

Softwarové požadavky
